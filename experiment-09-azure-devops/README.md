# Experiment 09: Introduction to Azure DevOps

## Aim
Understand Azure DevOps Services, set up account, create project, explore Boards, Repos, Pipelines.

## Theory
Azure DevOps provides: Boards (Agile), Repos (Git), Pipelines (CI/CD), Test Plans, Artifacts.

## Files
- `azure-pipelines.yml` — Basic build pipeline

## Usage

### Setup Steps
1. Go to https://dev.azure.com
2. Create organization
3. Create project: DevOps-BCSL657D
4. Import repository from GitLab

### Create Pipeline
1. Pipelines → New Pipeline
2. Select Azure Repos Git
3. Select repository
4. Select Existing YAML file
5. Select: experiment-09-azure-devops/azure-pipelines.yml
6. Run pipeline

## Azure DevOps Services
| Service | Description |
|---------|-------------|
| Boards | Agile planning |
| Repos | Git hosting |
| Pipelines | CI/CD |
| Test Plans | Testing |
| Artifacts | Package management |
